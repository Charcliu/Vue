### Vue 表单输入绑定

> 通过使用v-model指令绑定到input textarea checkbox radio实现双向绑定

> 表单输入修饰符

| 修饰符  | 作用                                                                    |
| ------- | ----------------------------------------------------------------------- |
| .lazy   | lazy 修饰符修改 input 触发为 change 触发                                |
| .number | number 修饰符可以将返回结果自动转化为 Number 类型，否则就是 String 类型 |
| .trim   | trim 过滤首尾空格符                                                     |
